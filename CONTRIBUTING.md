# Contributing

Thanks for helping with Speakeasy Windows OSS.

## Project Boundary

This repository is for the open-source Windows version only. Do not copy files from private product repositories, macOS-specific app code, local settings, packaged builds, recordings, logs, or credentials.

## Development

Use the setup and check commands in `README.md`. Windows-native behavior needs real Windows testing before release.

## Pull Requests

- Keep changes focused and easy to review.
- Include tests or manual checks in proportion to the change.
- Do not add accounts, sync, dictation history, or broader product workflows without prior project discussion.
- Never commit secrets or real environment values.
